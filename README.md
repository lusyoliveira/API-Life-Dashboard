## API 
- Este projeto tem proposito que criar uma API para comunicação com a aplicação front-end [Lifedashboard](https://github.com/lusyoliveira/Life-Dashboard)

## Rodando localmente
Clone o projeto e entre no diretório

`git clone <link_do_projeto>`
`cd API-life-dashboard`

### Instale as dependências
 `npm install`

## Inicie o servidor
npm run dev
